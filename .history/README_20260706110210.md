# Dice-Rolling Project Structure
DiceGame(tk.Tk): The main application class inheriting from Tkinter's root window.

setup_ui(): Constructs the control panel, including the session counter, dice selector, and roll button.

roll_dice(): Handles the game logic, updates the dice values using the random module, and increments the session counter.

draw_dice(): Manages the grid layout for displaying the dice.

create_die_canvas(): A helper function that mathematically draws a die and its dots on a Tkinter Canvas based on the rolled value.
